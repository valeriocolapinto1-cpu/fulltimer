# fulltimer
Rubik’s Cube Timer for mobile built entirely with **Claude’s AI system**, in **DART and FLUTTER**.
*(The first versions are made with Sonnet 4.6)*


# usage
To use it download the **Dart and Flutter** extensions on **Visual Studio Code**.
Download Android Studio and create a Virtual Machine based on Google Pixel 6.

Download the source code, add it to a **New Folder** and run
> flutter pub get
> 
To download the request sources. To open it, firstly start your **VM** on **Android Studio**, then run
> flutter run
>
